# NetSpy - Network Scanner & Monitor

Утилита для сетевого аудита: многопоточное TCP-сканирование, UDP-пинг, мониторинг доступности узлов. Написана на Python с использованием `scapy` и `tqdm`.

## Возможности
- Быстрое TCP-сканирование портов (1-65535) с прогресс-баром
- UDP-пинг для проверки доступности хоста
- Пинг мониторинг с логированием в файл

## Установка
```bash
git clone https://github.com/stavrogin1821/netspy.git
cd netspy
pip install scapy tqdm
```


## Использование
Утилита имеет три режима работы: scan, udp, monitor. Они вызываются как подкоманды. Справку по каждому режиму можно получить, добавив -h после подкоманды

### TCP-сканирование портов
```bash
python netspy.py scan <IP-адрес> [опции] 
```
Опции: 
+ -p, --ports – диапазон портов (по умолчанию 1-1024). Формат: начало-конец.

+ -w, --workers – количество потоков (по умолчанию 50).

### UDP-пинг
```bash
python netspy.py udp <IP-адрес> [--port PORT] 
```
Опции: 
+ --port – UDP-порт назначения (по умолчанию 33434).

### Мониторинг доступности
```bash
python netspy.py monitor <хост1> [хост2 ...] [опции] 
```
Опции: 

+ -i, --interval – интервал между проверками в секундах (по умолчанию 5).

+ -l, --log – путь к лог-файлу (по умолчанию monitor.log).
